# 数组
+ **「静态数组」就是一块连续的内存空间，我们可以通过索引来访问这块内存空间中的元素，这才是数组的原始形态。**
+ 动态数组 - 静态数组的基础上加上了API

### 静态数组
+ 数组名会自动衰退为指向首元素的指针（int*），所以数组首地址就是第一个元素的地址
+ 数组的元素类型决定了每个元素占用的存储空间大小
    + Byte 是内存与存储的基本寻址单位，也是大多数数据类型计算大小的基础。
    + 1 字节（Byte） = 8 比特（bit），bit 是最小的二进制单位，只能表示 0 或 1。
    + sizeof(int)（4字节）， sizeof(double)（8字节）
+ 数组的随机访问时间复杂度是O(1)

### **数据结构的职责就是增删查改**
+ 增
    1. 数组末尾追加（append）- O(1)
    2. 数组中间插入（insert）- O(n)
    3. 数组空间已满 - 连续内存必须一次性分配，分配完了之后就不能随意增减了 -> 扩容 O(n)
+ 删
    1. 删除末尾元素 - O(1)
    2. 删除中间元素 - O(n)
+ 查 
    1. 给出索引 - O(1)
    2. 遍历查找 - O(n)
+ 改 - O(1)

### 动态数组
+ **动态数组底层还是静态数组，只是自动帮我们进行数组空间的扩缩容，并把增删查改操作进行了封装，让我们使用起来更方便而已。**
+ 数组随机访问的超能力源于数组连续的内存空间，而连续的内存空间就不可避免地面对数据搬移和扩缩容的问题。

```cpp
vector<int> arr;
arr.push_back(i);
arr.insert(arr.begin() + 2, 666);
arr.pop_back();
arr.erase(arr.begin() + 2);
int index = find(arr.begin(), arr.end(), 666) - arr.begin();
```

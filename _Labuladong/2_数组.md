# 数组
## 双指针技巧秒杀七道数组题目
+ 双指针分为两类：左右指针和快慢指针
  + 左右指针，两个指针相向/相背而行
  + 快慢指针，两个指针同向而行，一快一慢
### 快慢指针技巧
1. 原地修改
   + [Leecode] 26. Remove Duplicates from Sorted Array
   + slow fast 双指针，当fast和slow不同的时候直接进行替换
   + [Leecode] 83. Remove Duplicates from Sorted List
   + 解法同上
     > 没有head的链表结点，如果是有gc的语言会自动回收，没有的话需要手动回收
   + [Leecode] 27. Remove Element
   + 对数组中的指定元素进行「原地删除」
     > 和 Lc26 去重的差异，slow++的位置不同 → 在于是否需要保留第一个元素
   + [Leecode] 283. Move Zeroes
   + 解法1 双指针，fast负责遍历全array，slow保证第一个非0的位置
   + 解法2 覆写法，先删除到末尾，再将结尾全转化为0

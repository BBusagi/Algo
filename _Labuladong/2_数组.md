### 左右指针的常用算法
1. 二分查找
``` python
def binarySearch(nums: List[int], target: int) -> int:
    # 一左一右两个指针相向而行
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (right + left) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        elif nums[mid] > target:
            right = mid - 1
    return -1
```

4. 回文串判断
```python
def isPalindrome(s: str) -> bool:
    # 一左一右两个指针相向而行
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True
```

## 滑动窗口算法核心代码模板
+ 滑动窗口算法技巧主要用来解决子数组问题，比如让你寻找符合某个条件的最长/最短子数组。
+ 类型为如何聪明地穷举
+ （穷举）暴力for循环解法，双层for循环，复杂度为n^2
+ （非穷举）滑动窗口的复杂度是n，区别在于**指针不回退**
+ **滑动窗口并不能穷举出所有子串**
+ 辨别的关键问题
  1. 什么时候扩大窗口？
  2. 什么时候缩小窗口？
  3. 什么时候更新结果？
+ 代码框架，包含输出debug位置，默写出来如下框架然后改三个地方就行

+ 解法2 滑动窗口

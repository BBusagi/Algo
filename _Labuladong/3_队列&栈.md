# 队列/栈
## 队列实现栈以及栈实现队列
### 1. 用栈实现队列
+ [Leecode] 232. Implement Queue using Stacks
  + queue = (stack2][stack1)
  + 新数据只进s1，读数据只从s2，**负负得正**
  + 只有s2中为空的时候，才会将s1倒进去
  + pop/peek最坏时间复杂度为On，均摊时间复杂度为O1
  + 所以复杂度都是O1

### 2. 用队列实现栈
+ [Leecode] 225. Implement Stack using Queues
  + 推荐使用deque()
  + [] → pop(0), deque() → popleft()
  + pop的时候，旋转n-1次，但是需要注意更新top值
  + pop复杂度为On，其他都是O1

## 单调栈
+ 单调栈保持栈内元素有序，单调递增或单调递减
+ 与heap相比，用法单一
### 例「下一个更大元素」
+ 解法1 递减栈，但是需要记录索引位置
+ 解法2 转化为身高问题
  + 检索后查看每个数据的后面的第一个高个子，并且将矮个子剔除
  + 虽然是双层循环，但是由于栈内已经经过了筛选，所以复杂度为On
+ 两套模板
  + 正向（从左向右）存索引，"我现在不知道，但是之后有人帮我算"
  + 反向（从右向左）存候选答案，"先将右侧整理好，直接查找"

+ [Leecode] 496. Next Greater Element I
+ 解法1 穷举解法
+ 解法2 单调递减栈
  + 单调栈先获取greaterlist
  + 然后利用非重复子集的特性，直接将数字做键值配对

# 队列/栈
## 队列实现栈以及栈实现队列
### 1. 用栈实现队列
+ [Leecode] 232. Implement Queue using Stacks
  + queue = (stack2][stack1)
  + 新数据只进s1，读数据只从s2，**负负得正**
  + 只有s2中为空的时候，才会将s1倒进去
  + pop/peek最坏时间复杂度为On，均摊时间复杂度为O1
  + 所以复杂度都是O1

### 2. 用队列实现栈
+ [Leecode] 225. Implement Stack using Queues
  + 推荐使用deque()
  + [] → pop(0), deque() → popleft()
  + pop的时候，旋转n-1次，但是需要注意更新top值
  + pop复杂度为On，其他都是O1

# 二叉堆原理
## 二叉堆核心原理及可视化
+ **二叉堆是一种能够动态排序的数据结构，是二叉树结构的延伸。**
+ 主要操作，sink（下沉）和 swim（上浮），维护二叉堆
+ 主要应用
  1. 数据结构优先级队列（Priority Queue）
  2. 排序方法堆排序（Heap Sort）
+ 二叉堆 vs BST
  + 二叉堆 局部有序，**找最值**O1
  + BST 全局有序，排序结构

### 二叉堆的性质
+ 二叉堆是一种动态排序的数据结构，即添加或删除元素之后会自动调整。
+ 二叉堆是优先级队列的底层实现，优先级队列能做到的，BST也能做到，但API和实现较为简单，能解决会优先选用
+ 任意节点的值，都必须大于等于（大顶堆）/小于等于（小顶堆）其左右子树所有节点的值
+ 二叉堆的左右子堆（子树）也是一个二叉堆

## python常用语法
### 取模
a, b  = divmod(a, b) //(a // b, a % b)

### 平方再排序
+ result = sorted(x*x for x in nums)

### sort API
+ list.sort() / list.sort(reverse=True)
+ 原地修改，返回None
+ sorted(nums) / sorted(nums, reverse=True)
+ 非原地，返回新列表
+ 复杂度 O(nlogn) 

### reverse API
+ list.reverse()
+ 原地修改，返回None
+ reversed()
+ 返回 iterator，不修改原列表

### 大写字母转化为小写字母
+ c.lower()

### 快速检测是否为字母或者数字
+ c.isalnum()

### 生成n*n矩阵
+ matrix = [[0] * n for _ in range(n)]

### 空格处理
+ 按照空格分割并处理多余空格：string.split()

### 反转拼接
+ 常用反转拼接: " ".join(reversed(words))

### 找最值
+  max(a, b, c, key = len)

### 方法定义
```C#
private int Function(int x) {
    return 100;
}

```

``` python
# 自动定义类型
def function(x):
    return 100

# 指定类型
def function(x: int) -> int:
    return 100
```

### 优先级队列
```python
import heapq
heap = []
# 插入元素
heappush(heap, x)
# 弹出元素
heappop(heap)
#查看最小元素（不弹出）
heap[0]
# 原地建堆
heapify(list)
# 只有最小堆，最大堆 → 优先级取负
push (-priority, value)
# ？？？ 元素比较规则
(priority, data)
```

```c#
var pq = new PriorityQueue<TElement, TPriority>();
// 插入
Enqueue(element, priority)
// 弹出
Dequeue()
// 查看最小元素
Peek()
// 元素数量
Count
// 安全弹出
TryDequeue(out e, out p)
```

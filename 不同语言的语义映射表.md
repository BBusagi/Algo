### 优先级队列
```python
import heapq
heap = []
# 插入元素
heappush(heap, x)
# 弹出元素
heappop(heap)
#查看最小元素（不弹出）
heap[0]
# 原地建堆
heapify(list)
# 只有最小堆，最大堆 → 优先级取负
push (-priority, value)
# ？？？ 元素比较规则
(priority, data)
```

```c#
var pq = new PriorityQueue<TElement, TPriority>();
// 插入
Enqueue(element, priority)
// 弹出
Dequeue()
// 查看最小元素
Peek()
// 元素数量
Count
// 安全弹出
TryDequeue(out e, out p)
```
